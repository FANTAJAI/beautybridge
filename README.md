# BeautyBridge — Sunmi Skincare

> Plateforme B2B connectant les grossistes nigérians aux acheteurs européens.

## 🚀 Déployer sur GitHub Pages (gratuit)

### Étape 1 — Créer le repo
1. Va sur [github.com](https://github.com) → **New repository**
2. Nomme-le : `beautybridge` (ou ce que tu veux)
3. Mets-le en **Public**
4. Clique **Create repository**

### Étape 2 — Uploader les fichiers
1. Dans ton repo → clique **"uploading an existing file"**
2. Glisse `index.html` et ce `README.md`
3. Clique **Commit changes**

### Étape 3 — Activer GitHub Pages
1. Ton repo → **Settings** → **Pages** (menu gauche)
2. Source : **Deploy from a branch**
3. Branch : **main** → dossier : **/ (root)**
4. Clique **Save**
5. ⏳ Attends 1–2 minutes
6. Ton app est live sur : `https://TON-USERNAME.github.io/beautybridge`
